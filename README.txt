
Structure of the proogram:

1. Read Pyscf-formatted molecule and generate fcidump using input.py
2. Run qsharp to get the estimates for a single Trotter step
3. Use parser.py to convert data.txt generated by qsharp or Openfermion to data.csv
4. Use QREChem.py to read data.csv and hardware parameters file to generate output.csv file

How to use instructions:

conda activate qrechem
mkdir QREChem
cd QREChem
git clone
./main.csh
conda qrechem
