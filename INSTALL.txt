Installation instructions:

conda create --name qrechem python=3.7.0
conda activate qrechem
apt-get install cmake
wget https://repo.anaconda.com/archive/Anaconda3-2023.03-1-Linux-x86_64.sh
bash ./Anaconda3-2023.03-1-Linux-x86_64.sh
source ~/.bashrc
pip install pyscf==2.0.1
pip install pandas

The version requirement of .NET (https://learn.microsoft.com/en-us/azure/quantum/install-update-qdk?tabs=tabid-conda)
.NET version of 6.0 or above for QDK versions of 0.24.201332 and above
.NET version of 3.1 for older QDK version

mkdir -p $HOME/dotnet && tar zxf dotnet-sdk-6.0.408-linux-x64.tar.gz -C $HOME/dotnet
export DOTNET_ROOT=$HOME/dotnet
export PATH=$PATH:$HOME/dotnet
pip install qsharp
dotnet tool install --global Microsoft.Quantum.Chemistry.Tools
conda qrechem
